# FIT2179 Data Visualisation 2

Single-page Vega-Lite data story for FIT2179, hosted with GitHub Pages.

## Data

- `ndshs_alcohol_preferences_2022_2023.json`: AIHW National Drug Strategy Household Survey alcohol preference data, 2022-23. Source: https://www.aihw.gov.au/reports/illicit-use-of-drugs/national-drug-strategy-household-survey/data?&page=1
- `gabs_hottest_100_2025.json`: GABS Hottest 100 Australian Craft Beers 2025 results, scraped from the official public result page and structured for Vega-Lite. Source: https://gabshottest100.com/auresults/2025/
- `australian_states.geojson`: Australian state geometries from Rowan Hogan's public GeoJSON. Source: https://github.com/rowanhogan/australian-states/blob/master/states.geojson
